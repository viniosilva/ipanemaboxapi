# Ipanema Box API

Schedule, services and customers manager

## Tech Stack

- [Go](https://go.dev/)
- [Make](https://www.gnu.org/software/make/)
- [Project Layout](https://github.com/golang-standards/project-layout)

## Run Locally

Install dependencies

```bash
  make
```

Start the server

```bash
  make dev
```


## Running Tests

To run tests, run the following command

```bash
  # unit tests
  make test

  # e2e tests
  make test/e2e
```
